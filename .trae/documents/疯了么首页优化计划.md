# 《疯了么》首页优化计划

## [ ] 任务1: 新增每日签到排名系统
- **优先级**: P0
- **依赖**: 无
- **描述**:
  - 新增每日签到排名数据存储
  - 实现签到时自动计算并记录用户排名
  - 确保数据持久化
- **成功标准**:
  - 每次签到后能正确计算用户是今日第几个发疯的
  - 数据在本地存储中正确持久化
- **测试要求**:
  - `programmatic` TR-1.1: 签到后能读取到正确的排名数据
  - `human-judgement` TR-1.2: 排名数据在多次签到后正确递增
- **备注**:
  - 使用新的存储键来保存每日签到记录
  - 每天凌晨自动重置排名

## [ ] 任务2: 在签到成功弹窗中显示排名信息

  - 在语录弹窗中也显示排名信息
  - 确保排名信息清晰可见
- **成功标准**:
  - 签到成功后能看到"你是今日第X个发疯的"提示
  - 排名信息在弹窗中美观显示
- **测试要求**:
  - `programmatic` TR-2.1: 弹窗中显示的排名与存储的数据一致
  - `human-judgement` TR-2.2: 排名显示位置合理，字体大小适中
- **备注**:
  - 可以在认证弹窗的时间消息下方添加排名信息
