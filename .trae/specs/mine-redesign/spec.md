# 个人中心全方位重构 Spec

## Why
当前个人中心界面较为简单，功能模块分散，缺少统一的主题风格。需要进行全面重构，打造一个功能完善、界面美观、支持深色模式的个人中心。

## What Changes
- 全新的"疯友档案"主题视觉设计
- 支持浅色/深色/跟随系统 三种主题模式
- 功能模块重组，界面更清晰
- 新增更多疯度相关数据展示
- 优化徽章展示体系

## Impact
- Affected specs: 首页签到系统、个人中心、主题系统
- Affected code: pages/mine/index.vue, App.vue, styles/*.scss

## MODULE: 主题系统

### Requirement: 主题切换
系统 SHALL 提供浅色/深色/跟随系统三种主题模式。

#### Scenario: 用户切换主题
- **WHEN** 用户在设置中选择主题模式
- **THEN** 系统更新全局主题，所有页面同步更新

#### Scenario: 用户选择跟随系统
- **WHEN** 用户选择"跟随系统"选项
- **THEN** 系统自动检测系统主题，实时同步

## MODULE: 个人中心重构

### Requirement: 用户档案头部
系统 SHALL 在个人中心顶部展示用户档案信息。

#### Scenario: 用户查看个人中心
- **WHEN** 用户进入个人中心
- **THEN** 显示：疯度等级图标、等级名称、总疯度值、加入日期

### Requirement: 核心数据卡片
系统 SHALL 提供四个核心数据展示卡片。

| 数据项 | 说明 | 图标 |
|--------|------|------|
| 总疯度 | 累计获得的疯度值 | ⚡ |
| 连续签到 | 连续签到天数 | 🔥 |
| 累计签到 | 总签到天数 | 📅 |
| 疯批次数 | 疯情测试次数 | 🧠 |

### Requirement: 疯度等级进度
系统 SHALL 展示当前等级到下一等级的进度。

#### Scenario: 用户查看等级进度
- **WHEN** 用户查看个人中心
- **THEN** 显示当前等级进度条和距离下一等级的数值

### Requirement: 成就徽章墙
系统 SHALL 展示用户的成就徽章。

#### Scenario: 用户查看徽章
- **WHEN** 用户进入个人中心
- **THEN** 展示已解锁和未解锁的成就徽章

### Requirement: 功能菜单
系统 SHALL 提供主要功能入口。

| 功能 | 图标 | 说明 |
|------|------|------|
| 历史记录 | 📜 | 查看疯批历史 |
| 疯度排名 | 🏆 | 查看疯度排行榜 |
| 设置 | ⚙️ | 主题切换等功能 |
| 关于 | ℹ️ | 关于疯了么 |
| 意见反馈 | 💬 | 反馈问题 |

### Requirement: 主题设置入口
系统 SHALL 在设置中提供主题切换选项。

## ADDED Requirements

### Requirement: 疯度进度条
系统 SHALL 提供当前等级到下一等级的进度可视化。

#### Scenario: 用户查看等级进度
- **WHEN** 用户已解锁当前等级
- **THEN** 显示进度条和距离下一等级所需数值

## REMOVED Requirements
无

## 数据结构

### CheckinData
```typescript
interface CheckinData {
  lastDate: string          // 最后签到日期
  continuousDays: number    // 连续签到天数
  totalDays: number         // 累计签到天数
  crazyPoints: number       // 总疯度值
  joinDate: string          // 加入疯友联盟日期
}
```

### CrazyLevel
| 连续天数或疯度值 | 等级名称 | 图标 | 描述 |
|----------------|---------|------|------|
| <1天 或 0 | 潜水路人 | 🚶 | 还没开始发疯，快来报到 |
| ≥1天 或 ≥1 | 初入疯途 | 🌱 | 恭喜你，正式成为疯友 |
| ≥7天 或 ≥300 | 疯途新秀 | 🚀 | 踏上疯途，小有名气 |
| ≥30天 或 ≥1000 | 小疯初成 | 🌟 | 初具疯形，未来可期 |
| ≥100天 或 ≥3000 | 疯林盟主 | 🏆 | 一方霸主，疯名远扬 |
| ≥365天 或 ≥10000 | 疯神降临 | 👑 | 疯界天花板，疯得惊天动地 |
